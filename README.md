### What I learned

I learned how to add a color accent over an image using linear-gradient

```footer {
    width: 350px;
    height: 400px;
    margin-top: -68px;
    background: linear-gradient(
    rgba(77, 150, 168, 0.9),
    rgba(77, 150, 168, 0.9)),
    url('../assets/mobile/image-footer.jpg') 
    center / cover
    no-repeat;
}```
